# Product Overview

MyStockApp 是一個股票管理應用程式，提供基於 Web 的股票資訊管理與追蹤功能。

## Core Capabilities

- **Web 界面存取**：透過瀏覽器提供互動式股票管理功能
- **資料持久化**：使用 PostgreSQL 儲存股票相關資料
- **即時互動**：採用 Blazor Server 實現即時雙向通訊
- **跨環境部署**：支援本地開發與雲端環境（Heroku）部署

## Target Use Cases

- 個人股票組合管理與追蹤
- 股票資訊即時查詢與更新
- 多環境（開發、生產）的資料管理需求

## Value Proposition

採用 .NET 10 最新技術棧與 Blazor Server 架構，提供無需 JavaScript 框架即可實現的豐富互動體驗，並透過 Entity Framework Core 簡化資料存取邏輯，支援彈性的雲端與本地部署策略。
